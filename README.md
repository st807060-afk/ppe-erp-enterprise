# PPE ERP Enterprise

PPE ERP Enterprise (Petit Prix Déstockage ERP Enterprise) est un ERP moderne développé en Java 21 et Spring Boot.

## Technologies

### Backend
- Java 21
- Spring Boot 3
- Maven
- PostgreSQL
- Spring Security
- JWT
- Flyway
- Swagger / OpenAPI

### Frontend
- React
- TypeScript

### Infrastructure
- Docker
- Docker Compose

## Architecture

```
backend/
frontend/
database/
docker/
docs/
tests/
```
