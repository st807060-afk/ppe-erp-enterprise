# Guide utilisateur

Bienvenue dans **PPE ERP Enterprise**.

Ce guide présente les principales fonctionnalités du logiciel.

---

# Connexion

Accédez à l'application.

Saisissez :

- votre identifiant
- votre mot de passe

Cliquez sur **Connexion**.

---

# Tableau de bord

Le tableau de bord affiche :

- nombre de produits
- nombre de clients
- nombre de fournisseurs
- chiffre d'affaires (version future)
- alertes

---

# Produits

Depuis le menu **Produits** :

- afficher les produits
- ajouter un produit
- modifier un produit
- supprimer un produit

---

# Clients

Depuis le menu **Clients** :

- consulter les clients
- créer un client
- modifier un client
- supprimer un client

---

# Fournisseurs

Depuis le menu **Fournisseurs** :

- consulter les fournisseurs
- créer un fournisseur
- modifier un fournisseur
- supprimer un fournisseur

---

# Commandes

(Disponible à partir de la version 0.3.0)

- créer une commande
- ajouter des produits
- calcul automatique du total

---

# Stock

(Disponible à partir de la version 0.4.0)

- consulter le stock
- mouvements de stock
- alertes

---

# Facturation

(Disponible à partir de la version 0.5.0)

- création de facture
- génération PDF
- paiements

---
